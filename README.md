# ATS Resume Analyzer

An AI-powered ATS Resume Analyzer that evaluates resumes against job descriptions and provides optimization suggestions.

## Planned Features

- User Authentication
- Resume Upload
- Resume Parsing
- ATS Score Generation
- Skill Gap Analysis
- AI-Powered Suggestions
- Dashboard & Analytics

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- FastAPI
- Python

### Database
- PostgreSQL
